この旅行プランナーアプリ、TravelPlanerは、サポーターズのハッカソンプロジェクトとして5人チームのIM Technologyが開発しました。

使用言語

  • Java
  • HTML
  • CSS

開発環境

  • SpringBoot

データ環境

  • MySQL

チームメンバー

  • 小山田卓生 - プロジェクトリーダー: 旅行プランナーアプリの全体設計と管理を担当しました。
  • メンバー2 - フロントエンド開発者: ユーザーインターフェースとユーザーエクスペリエンスの設計を担当しました。
  • メンバー3 - バックエンド開発者: データベース設計とサーバーサイドロジックの実装を担当しました。
  • メンバー4 - デザイナー: アプリケーションのビジュアルデザインとブランド設定を担当しました。
  • メンバー5 - テスター: アプリケーションの品質保証とバグフィックスを担当しました。

プロジェクトについて

このプロジェクトは、旅行計画をより簡単に、そして楽しくすることを目指して開発されました。目的地を入力するだけで、食事、宿泊、観光スポットの推薦、日程調整、文化や習慣のアドバイス、天気情報、予算管理、地域情報など、旅行に必要な情報を一つのプラットフォームで提供します。
